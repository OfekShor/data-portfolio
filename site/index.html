Stock Portfolio Automation Dashboard

Track, Analyze, and Export stock data with ease – powered by Google Sheets, SQLite, and Tableau.

🔗 System Workflow

  1. Data Source: Google Sheets containing portfolio and stock price data
  2. Python Scripts: Automate syncing to SQLite and exporting to CSV
  3. Custom SQL: Write queries in query.sql or use saved templates
  4. Visualization: Load results into Tableau for interactive charts

📂 Project Structure

  • src/ – Python scripts for syncing, exporting, and querying
  • queries/ – Saved SQL query templates
  • csv_exports/ – Auto-exported CSVs for Tableau
  • query.sql – Last active SQL query
  • query_log.txt – Query history log

📊 Visualization with Tableau

Use the exported query_result.csv or any other CSVs to create dashboards in Tableau Public.

💡 Key Features

  • 🔄 Daily sync from Google Sheets
  • 🧠 Flexible SQL query system
  • 📈 Visual insights via Tableau
  • 📝 Automatic CSV generation
